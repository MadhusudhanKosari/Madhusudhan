# Portfolio Frontend

This is the frontend for the portfolio website, built with React + Vite + Tailwind CSS.

## Scripts

- `npm run dev`: Start development server.
- `npm run build`: Build for production.
- `npm run preview`: Preview production build.

## Key Components

- `Navbar`, `Footer`: Layout components.
- `ProjectCard`: Reusable card for displaying projects.
- `Pages`: Home, About, Projects, Research, Leadership, Resume, Contact.

## Data
Data is centralized in `src/data/` for easy updates without touching components.
